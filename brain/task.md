# Task: Create a Landing Page

## Objective
To create a visually appealing and functional landing page that adheres to the specified design guidelines (vibrant HSL colors, modern typography, glassmorphism).

## Plan

1.  **Research & Setup**:
    *   Examine existing project structure to identify suitable locations for new files (HTML, CSS, JS).
    *   Set up a basic HTML structure for the landing page.

2.  **Design & Development**:
    *   Implement the visual design using CSS, incorporating glassmorphism, vibrant HSL colors, and modern typography (Inter, Outfit).
    *   Create a responsive layout (mobile-first).
    *   Add subtle micro-interactions and animations.
    *   Generate a hero image using `generateImage`.

3.  **Verification**:
    *   Use the browser sub-agent to open the local HTML file and take a screenshot to verify the visual output.
    *   Generate a `walkthrough.md` with proof of completion.
