# Nexus OS Project Roadmap (D: Drive)

This project aim to build **Nexus OS**, a full-featured agentic AI.

## Directory Structure
- `brain/`: Persistent memory, task tracking, and state.
- `core/`: System prompts, orchestrator logic, and LLM interfaces.
- `tools/`: JavaScript implementations of file, terminal, browser, and image tools.
- `scratch/`: Temporary area for the agent to experiment.
